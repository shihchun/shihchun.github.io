Install NS3

Install the dependency of the NS3, I try to install all of them as I can. See the NS3 wiki.

gccxml

If you use the wsl (ubuntu 18.04), then you may have the problem with the gccxml. I just build it from source code in the official site, but the cmake command not works for me, I just google it, find the solution as below:

1
2
3

cmake -D CMAKE_C_FLAGS=-fgnu89-inline ../gccxml
make
sudo make install

Wireshark PCAP

If you use the wsl to do packet capture (PCAP), you can install the wireshark, and just add it to the windows env.

1

choco install wireshark

Then add to the env.

Then just use it in wsl by this way.

1
2

./waf --run sixth
wireshark.exe sixth.pcap
